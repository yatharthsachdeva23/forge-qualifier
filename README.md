## What This Does

This project demonstrates an end-to-end multi-agent AI orchestration loop built for the **Forge Sprint Qualifier Challenge**.

Two AI agents collaborate autonomously over Slack to write code, execute it, and report results:

- **Hermes** (Orchestrator): Reads the custom `web-scraper` skill, plans the task, and assigns work to the coding agent via `#agent-coder`.
- **OpenClaw** (Coder): Picks up the task from Slack, writes Python code, runs it locally, and posts structured status updates back to Slack.

## Architecture

```
You (Human)
    │
    ▼
Slack: #sprint-main
    │  @Hermes, execute the web-scraper skill
    ▼
Hermes Orchestrator (Ollama Cloud — Gemma4:31B)
    │  Plans task, loads SKILL.md
    ▼
Slack: #agent-coder
    │  @Forge Agent Gateway, write a Python scraper for 3 URLs
    ▼
OpenClaw Coder (OpenRouter — Cohere North Mini Code)
    │  Writes web_scraper.py, runs it locally
    ▼
scrape_results.json  ──────►  Slack Status Reply
    │
    ▼
Dashboard: forge-qualifier.vercel.app
```

## Scraper Output

The scraper extracts `<title>` and all `<h1>` through `<h6>` headings from:

- `https://news.ycombinator.com`
- `https://python.org`
- `https://github.com`
