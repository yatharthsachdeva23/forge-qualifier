# Project: Collaborative Kanban Board (V2)

## Quick Start
### 🛠️ Backend (Laravel)
1. Navigate to the backend directory:
   `cd backend`
2. Start the API server:
   `php artisan serve`
   *(Default URL: http://localhost:8000)*

### 💻 Frontend (React + Vite)
1. Navigate to the frontend directory:
   `cd frontend`
2. Install dependencies:
   `npm install`
3. Start the development server:
   `npm run dev`

## Project Structure
- `/backend`: Laravel REST API (SQLite)
- `/frontend`: React + Vite + Tailwind CSS
- `/config`: Redacted system configurations
- `/screenshots`: Visual evidence of implementation
- `/skills`: Status skill definitions
